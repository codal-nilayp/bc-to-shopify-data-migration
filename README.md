# ENV file 
- Create .env file
- Add below variables  with proper values
  - BC_STORE_HASH
  - BC_TOKEN
  - SHOPIFY_STORE
  - SHOPIFY_ADMIN_TOKEN
  - SHOPIFY_API_VERSION

# Steps 
- Install node.js with [nvm](https://github.com/creationix/nvm)
- Run `nvm use`
- Use [node 20](https://nodejs.org/en/about/releases/)
- Run `npm i`
- Run `node products.js`
